Fizz Buzz Exercise

Please enter a number in the "Entry" fields below ranging between 1 and 100.

- For each number that is a multiple of your first entry, the word "Fizz" will populate.

- For each number that is a multiple of your second entry, the word "Buzz" will populate.

- For each number that is a multiple of both entries, the word "FizzBuzz" will populate.

Give it a try!

//step2: make calculations with data
        function FizzBuzzA(value1, value2) {
            let returnValue = "";
            for (let i = 1; i <= 100; i++) {
                if (i % value1 == 0 && i % value2 == 0) {
                    returnValue += "FizzBuzz, ";
                }
                else if (i % value1 == 0) {
                    returnValue += "Fizz, ";
                }
                else if (i % value2 == 0) {
                    returnValue += "Buzz, ";
                }
                else {
                    returnValue += i + " " + ", ";
                }
            }
            return returnValue;
        }

Palindrome Exercise

This exercise is used to determine if the Users' input is a Palindrome.

A Palindrome is a word, phrase, or sequence that reads the same backwards as it does forward.

For example; dad, pop, and mom, are all Palindromes.

Give it a try!

$("#Palbtn").click(function () {
            var word = $("#PalInput").val();
            word = word.replace(/\W/g, "");
            word = word.toLowerCase();
            var listArray = word.split('')
            var flipArray = listArray.reverse();
            var wordFlip = flipArray.join('');

            if (word === wordFlip) {
                $("#PalOutput").text('Success! ' + word + ' is a Palindrome.')
            } else {
                $("#PalOutput").text('Sorry, ' + word + ' is not a Palindrome.')
            }
        });

        //Clearing the Input & Output fields

        $("#PalClear").click(function () {
            $("#PalInput").val('');
            $("#PalOutput").text('');
        });

Factorial Exercise

This exercise is used to determine the factorial of a given number.

A Factorial is the product of an integer and all the integers below it.

For example, the factorial of 4 is 24...(4*3*2*1) = 24

Give it a try!

//Get User Data
        $("#Factorialbtn").click(function () {
            var number = Number($("#FactInput").val());

            //Evaluate Data
            if (number >= 171) {
                alert("Please enter a valid number between 1 and 171.");
                $("#FactInput").val("");
                return;
            }

            //Perform Factorial
            for (var Factorial = number - 1; Factorial > 1; Factorial--) {
                number *= Factorial;
            }

            //Show Result
            $("#FactOutput").text(number);
        });

        //Clear Input and Output Fields
        $("#FactorialClear").click(function () {
            $("#FactInput").val("");
            $("#FactOutput").text("");
        });
